## 怎么使用

```bash

  # download the zip
  $ git clone https://github.com/reng99/webapp.git

  # enter the root catalogue
  $ cd webapp

  # install dependencies
  $ npm install

  # dev mode
  $ npm run dev
  # automally show in browser whick address is localhost:8000

  # dev mode
  $ npm run build

```
